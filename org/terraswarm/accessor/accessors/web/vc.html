The Accessors Repository

This accessors repository is a Git repository

The repository includes a library of accessors, some swarmlet hosts (including a Browser and Node.js host), and the https://ptolemy.berkeley.edu/accessors/ web page contents. It does not include the Cape Code swarmlet host (which uses its accessor library), nor the accessors wiki pages, which are used to document the accessor API and swarmlet hosts, nor the auto-generated JSDoc JavaScript code documentation.

Note that CapeCode, the Ptolemy II Accessors development environment, includes a copy of the accessors tree, so with CapeCode, it is not necessary to check out a separate copy of the accessors repo./

To check out the repo, use 

	git clone https://github.com/icyphy/accessors

The JSDoc documentation files are updated by a Jenkins job (see JSDoc Continuous Integration). Note that Jenkins job is not world readable, but the details are world readable.
